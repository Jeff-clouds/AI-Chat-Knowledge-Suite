# v1.1.0: Pro 授权码与勾选局部导出

## 主要变化

- 新增本地 Pro 授权码激活。
- 新增“选择问题组”模式。
- 新增 Pro 功能“导出选中内容”。
- 局部导出按问题组工作：勾选问题后，会导出该问题及其对应完整答案。
- 完整 Markdown 导出、大纲导航、阅读位置高亮继续免费。

## 授权说明

- 第一版不需要账号，不绑定机器。
- 授权码在扩展本地验签。
- 授权码中保留 licenseId、orderId、emailHash、plan、issuedAt，方便后续迁移到更严格的授权服务。
- 购买交付第一阶段采用半自动流程：确认外部订单后，使用本地脚本生成授权码并交付给用户。

## 验证重点

- 未激活 Pro 时，完整导出仍可用。
- 未激活 Pro 时，导出选中内容会提示激活。
- 激活有效授权码后，可勾选多个问题组并导出局部 Markdown。
- 选择模式不应破坏大纲跳转、展开折叠和阅读高亮。
