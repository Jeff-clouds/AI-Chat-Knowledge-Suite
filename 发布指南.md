```bash
gh release create v1.6.0 \
  --title "v1.6.0: ChatGPT 风格优化" \

```

或者使用以下命令：

```bash
gh release create v1.6.0 \
  --title "v1.6.0: ChatGPT 风格优化" \
  --notes "## 主要改进

- ✅ 采用 ChatGPT 设计风格：深色/浅色双主题支持
- ✅ 优化底色块缩进：使用 margin-left 实现整体缩进
- ✅ 修复目录收起动画：收起时下面的目录会移动上来
- ✅ 重构 CSS 变量系统：添加 --bg-item-border 等变量
- ✅ 优化响应式设计：两个断点适配不同屏幕

## 技术改进

- 使用 height 动画替代 max-height
- 使用 !important 确保 CSS 类覆盖内联样式
- 使用强制重绘确保动画生效
- 统一 toggle 逻辑

1. 下载 ai-chat-outline-v1.6.0.zip
2. 解压到本地文件夹
3. 打开 Chrome 扩展管理页面
4. 启用开发者模式
5. 加载已解压的扩展程序" \
  ai-chat-outline-v1.6.0.zip
```
