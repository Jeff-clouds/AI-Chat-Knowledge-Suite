# v2.0.0: AI Chat Export Pro

## 主要变化

- 对外名称统一为 **AI Chat Export Pro**。
- 明确 v2.0.0 是原 **AI Chat Outline** 的迭代合并版。
- 保留侧边栏大纲导航、阅读位置高亮和快捷键。
- 合并完整 Markdown 导出能力。
- 保留 Pro 授权码与勾选问题组局部导出能力。

## 发布定位

AI Chat Export Pro 面向 ChatGPT、DeepSeek、Gemini、Kimi、豆包、Grok、腾讯元宝等 AI 对话页面，帮助用户浏览长对话并沉淀 Markdown 知识材料。

## 安装测试

1. 下载 `ai-chat-export-pro-v2.0.0.zip`。
2. 解压到本地目录。
3. 打开 Chrome/Edge 扩展管理页面。
4. 开启开发者模式。
5. 选择“加载已解压的扩展程序”，加载解压后的目录。
