# v1.1.2: Streamlined Pro export layout

## 主要变化

- 普通用户顶部显示当前网站，第二行提示 Pro 可选择部分导出，并提供“激活Pro”入口。
- Pro 用户第二行显示“Pro用户”，右侧提供“部分导出”入口。
- Pro 部分导出模式中，右侧按钮切换为“退出选择模式”，问题组显示 checkbox。
- 底部固定栏统一保留“收起所有”和导出按钮。
- 选择模式下底部右侧按钮改为“导出已选对话”，未选择问题组时保持禁用。

## 设计说明

- 局部导出仍按问题组工作：勾选问题后导出该问题及其对应完整答案。
- 侧边栏顶部不再堆叠完整导出、选择模式、购买、激活等多枚按钮，让大纲阅读区域更稳定。
- 完整导出仍为免费核心功能；局部导出继续由 Pro 授权码解锁。

## 验证重点

- 免费用户可以正常导出完整对话，并通过“激活Pro”输入授权码。
- Pro 用户可以进入和退出部分导出模式。
- 勾选问题组后，“导出已选对话”从禁用变为可用。
- 展开/收起、大纲点击跳转和完整导出不受本次布局调整影响。
